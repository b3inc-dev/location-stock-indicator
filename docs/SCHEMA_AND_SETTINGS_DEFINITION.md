# スキーマ・設定の定義（種類統一・ボタン・区切り・項目数）

ボタンは「種類」で選択。**文字サイズ・間隔・余白・幅は range（レンジスライダー）** でプレビューを見ながら調整しやすくし、ボタン表示時のレイアウト調整を追加したうえで、**スニペットの調整箇所（ジャンル）・項目数** と **管理画面の項目** を一覧にした定義です。

---

## 1. スキーマの型の統一方針

**文字サイズ・間隔・余白・幅** はプレビューを見ながら細かく合わせたいため **range（レンジスライダー）** にし、他は select / color で統一します。

| 設定の種類 | 使うスキーマ型 | 値の例・備考 |
|------------|----------------|--------------|
| **揃え・配置** | `select` | left / center / right |
| **文字サイズ** | `range` | スライダーで調整しやすい。単位は px または %（スニペット側で em に換算しても可） |
| **間隔・余白** | `range` | スライダーで調整しやすい。単位は px で統一（min/max/step を項目ごとに定義） |
| **幅** | `range` | 最大幅など。0 で「100%」、正の値で px。スライダーで直感的に調整 |
| **枠線** | `select` | none / rows / full（現状どおり） |
| **色** | `color` | そのまま（テーマごとに変えたいため） |
| **ON/OFF** | `checkbox` または 管理画面側で一元化 | - |
| **文言** | 管理画面（在庫表示設定） | - |

※ 文字サイズ・間隔・幅は **range にするとカスタマイザーでプレビューを見ながら微調整しやすい**。`text`（"0.95em" などの直接入力）は使わず、range の min/max/step で範囲を決める。

## 2. ボタン（店舗受け取り）の定義

### 2.1 ボタンは「種類」＋「背景色・文字色」＋「余白」で調整

- **種類**: `order_pick_btn_style`（select）で **7種類**：（ソリッド / アウトライン）×（スクエア / ラウンド / ピル）＋ ゴースト。
- **色**: `order_pick_btn_background`（color）・`order_pick_btn_color`（color）で背景と文字色。未指定時は #f5f5f5 / #333。
- **余白**: `order_pick_btn_padding`（range）でボタンの文字まわりの余白（px）。角丸・形状は種類に含める。

| 値（id） | ラベル | 内容 |
|----------|--------|------|
| solid_square | ソリッド×スクエア | 塗りつぶし・角丸 0 |
| solid_round | ソリッド×ラウンド | 塗りつぶし・角丸 8px |
| solid_pill | ソリッド×ピル | 塗りつぶし・角丸 999px |
| outline_square | アウトライン×スクエア | 枠線のみ・角丸 0 |
| outline_round | アウトライン×ラウンド | 枠線のみ・角丸 8px |
| outline_pill | アウトライン×ピル | 枠線のみ・角丸 999px |
| ghost | ゴースト | 背景なし・枠なし・文字色のみ |

※ テーブル時の**ロケーション列の幅（%）**は `location_col_width` のまま。ボタン表示時も変更しない（常に効く）。**ボタン列の幅**だけ `order_pick_col_width`（range・%）で指定。

### 2.2 ボタン表示時のレイアウト調整（別途）

| 項目（id） | スキーマ型 | 説明 |
|------------|------------|------|
| **ボタン列の幅（テーブル時・%）** | range | `order_pick_col_width`。15〜45%、default 25。テーブルでボタン表示時のみ。区切り線は設けない。 |
| **ボタンの左右位置** | select | セル内・リスト・2行いずれでも反映。left / center / right |

## 3. 文字サイズ・間隔・幅の range（レンジスライダー）の共通定義

**文字サイズ・間隔・余白・幅** は **range** にし、プレビューを見ながらスライダーで調整しやすくします。単位は **px** で揃え、min / max / step を項目ごとに定義します。

### 3.1 文字サイズ（range）

| 項目例 | min | max | step | default | 備考 |
|--------|-----|-----|------|---------|------|
| ブロックの文字サイズ（font_size） | 12 | 24 | 1 | 16 | px。スニペット側で em に換算しても可 |
| エリア見出しの文字サイズ | 8 | 20 | 1 | 14 | px |

※ ボタンの文字サイズは「ボタンの種類」に含め、個別 range は持たない。

### 3.2 間隔・余白（range）

| 項目例 | min | max | step | default | 備考 |
|--------|-----|-----|------|---------|------|
| タイトル下マージン | 0 | 40 | 2 | 8 | px |
| ブロック上マージン | 0 | 80 | 2 | 0 | px |
| ブロック下マージン | 0 | 80 | 2 | 16 | px |
| 行の上下余白 | 0 | 32 | 2 | 8 | px。**注意書きとリストの間隔も同じ値で反映**（統一） |
| 行の左右余白 | 0 | 32 | 2 | 12 | px。テーブル・リストのセル/行の左右パディング（必要に応じて指定） |

→ 単位は **px** で統一。注意書きとリストの間隔は「行の上下余白」と同一項目で兼ねる。

### 3.3 幅（range）

| 項目例 | min | max | step | default | 備考 |
|--------|-----|-----|------|---------|------|
| 最大幅（content_max_width） | 0 | 1200 | 20 | 600 | px。0 のときは 100%（横幅いっぱい） |
| ロケーション列の幅（%） | 30 | 80 | 5 | 70 | テーブル時。**ボタン表示時も同じ値で効く**（変更しない） |
| ボタン列の幅（%） | 15 | 45 | 5 | 25 | テーブルでボタン表示時のみ。`order_pick_col_width` |
| ボタンの文字余白（px） | 0 | 24 | 2 | 8 | `order_pick_btn_padding` |

→ 最大幅だけ 0 = 100% の特別扱い。それ以外は range の数値をそのまま使う。

## 4.2 タイトル・リスト・凡例・注意書き・ボタン：統一 vs 独立

「同じ 1 本の設定でまとめて変える」か「それぞれ別項目で調整する」かを、**文字サイズ・色・間隔・揃え** ごとに整理しました。

### 統一した方がいいもの（一括で調整）

| 対象 | 項目 | 理由 |
|------|------|------|
| **リスト・凡例・注意書きの文字サイズ** | **ブロックの文字サイズ（font_size）1 本で統一** | 本文まわりを同じ大きさにすると読みやすく、スキーマ項目も増やさずに済む。凡例・注意書きを「本文より少し小さく」したい場合は、同じ font_size のままでもレイアウト上は許容できることが多い。 |
| **ボタンの見た目** | **種類（order_pick_btn_style）＋背景色・文字色** | 種類で角丸・枠の有無を選択。背景・文字色は別項目で指定可能。 |

→ **文字サイズ**は「ブロックの文字サイズ」で **リスト・凡例・注意書き** をまとめて変える。**タイトル・エリア見出し・ボタン** は下記のとおり独立。

### 独立させた方がいいもの（個別に調整）

| 対象 | 項目 | 理由 |
|------|------|------|
| **タイトル** | 文言・揃え・**文字サイズ**・下マージン・色 | タイトルは本文より大きく・目立たせたいことが多く、ブロックの文字サイズとは別にしたい。色や余白もタイトル専用にしたいことがある。 |
| **凡例** | 位置・揃え・**色** | 凡例だけ控えめの色にしたい場合がある。文字サイズは上記のとおりブロックに統一してよい。 |
| **注意書き** | 位置・揃え・**色** | 補足なので色を薄くしたい。**リストとの間隔は行の上下余白（row_padding_y）で統一**。文字サイズはブロックに統一。 |
| **リスト（行）** | 行の余白・列幅・枠線・**行内テキスト・マークの色** | 行の余白や枠線はリスト専用。在庫ステータスごとの色はリスト固有なので独立。 |
| **エリア見出し** | 背景色・文字色・**文字サイズ**・余白 | エリア単位の見出しなので、ブロック本文とは別のトーンにすることが多い。 |
| **ボタン** | 種類（7種）・背景色・文字色・余白・列幅・位置 | 種類で形状。色・余白・列幅（%）は range/color で独立。区切り線は設けない。 |

### 一覧表（パーツ別）

| パーツ | 文字サイズ | 色 | 間隔・余白 | 揃え・位置 |
|--------|------------|-----|------------|------------|
| **タイトル** | 独立（タイトル用 range） | 独立 | 独立（タイトル下） | 独立 |
| **リスト** | **統一**（ブロックの font_size） | 独立（行内・マークの色） | 独立（行の余白） | 独立（ロケーション・在庫の揃え） |
| **凡例** | **統一**（ブロックの font_size） | 独立 | （位置は select で上/下） | 独立 |
| **注意書き** | **統一**（ブロックの font_size） | 独立 | **行の上下余白で統一** | 独立 |
| **ボタン** | 種類に含める（個別項目なし） | 種類に含める | 種類に含める | 独立（左右位置） |

→ この結果、**文字サイズ** は「ブロックの文字サイズ」＋「タイトルの文字サイズ」＋「エリア見出しの文字サイズ」の **3 本** にすると、統一と独立のバランスが取りやすい（ボタンは種類内で固定）。  
※ 項目数は **interactive 41**。header / paragraph は別カウント（non-interactive）。

### 4.2 スニペット側で参照する「塊」のまとめ

| 調整箇所 | 内容 |
|----------|------|
| タイトル・ブロック全体 | タイトル文言・揃え・間隔・色、レイアウト、揃え、文字サイズ、ブロック余白・幅・配置 |
| 凡例・注意書きの見た目 | 凡例の位置・揃え・色、注意書きの位置・揃え・間隔・色（表示ON/OFF・文言は app） |
| 行・枠線・色 | 行の間隔、列幅、枠線スタイル・太さ・色、ステータス色、アコーディオンマーク |
| エリア見出し | 背景色・文字色・文字サイズ・余白（select で統一） |
| 店舗受け取りボタン | 種類（スタイル）、ボタン表示時の列幅・区切り線・ボタン位置 |
| その他 | 追加CSS |

## 7. まとめ

| 対象 | 内容 |
|------|------|
| **ボタン** | 種類 7 種（ソリッド/アウトライン×スクエア/ラウンド/ピル、ゴースト）＋背景色・文字色（color）＋余白（range）。区切り線は設けず、ボタン列幅は range（%）で指定。 |

| **スニペットの調整箇所** | 6 ブロック（タイトル・ブロック全体／凡例・注意書き／行・枠線・色／**エリア見出しのデザイン**（開閉マーク含む）／店舗受け取りボタン／その他）。テーマ側 interactive 41 項目。 |
| **管理画面（在庫表示設定）** | 閾値・マーク・在庫数表示・ステータスラベル・メッセージ・注意書き（文言＋表示ON/OFF）・凡例表示ON/OFF・行の表示内容・リスト区切り文字。 |
| **文字サイズ・間隔・幅** | カスタマイザーでは **range（レンジスライダー）** で調整しやすくする。単位は px（幅は 0 で 100%）。 |

実装時は、この定義に合わせてスキーマの id と range の min/max/step を揃え、スニペット側でボタン種類ごとのスタイル定義を行います。文字サイズ・間隔・幅は range の値をそのまま px（または %）として適用します。
